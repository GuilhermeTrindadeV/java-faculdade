# API de Tarefas

API RESTful para gerenciamento de tarefas desenvolvida em **Java com Spring Boot**.

## Funcionalidades

- Criar tarefa (POST)
- Listar todas as tarefas (GET)
- Buscar tarefa por ID (GET)
- Atualizar tarefa (PUT)
- Remover tarefa (DELETE)

## Tecnologias

- Java 17  
- Spring Boot 3  
- Spring Data JPA  
- MySQL
